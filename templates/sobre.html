Início Sobre

Sobre o projeto:

Scraping com Selenium Desenvolvi esse projeto com o objetivo de estudar mais sobre o metodo scraper junto com o selenium, ultimamente tenho usado outras linguagens de programação no meu trabalho, mais confesso que pytho sempre vai ter um lugar no meu coração, e por conta disso continuo a estudar mais e mais frameworks e bibliotecas de python, por puro entusiamos!

Este projeto possui uma interface simples, da qual desenvolvi com a ajuda do v0.dev Apesar de básica, ela facilita o entendimento e a visualização do que o meu código se propõe a entregar.

Principais Características:

  • Interface limpa e intuitiva para facilitar a experiência do usuário
  • Sistema de busca otimizado
  • Design responsivo que se adapta a diferentes dispositivos
  • Código HTML puro para máxima compatibilidade e desempenho
  • Fácil integração com outros sistemas e plataformas
